# 結局 key-art(Ending hero art)

一場遊戲的終幕。[EndingsModal.tsx](../../src/ui/components/EndingsModal.tsx) 頂部顯示一張
16:9 大圖 —— **有圖用圖、缺圖退回原本的程序水墨 SVG**(放檔即生效,不崩)。

| 用途 | 檔名 | 缺檔時 |
|---|---|---|
| **結局圖** | `public/endings/<kind>.jpg` | 退回該結局的 SVG 水墨小景(不變) |

- 規格:**16:9,~1400px 寬,JPEG q≈82**。壓縮(macOS 免裝工具):
  `sips -s format jpeg -s formatOptions 82 --resampleWidth 1400 in.png --out <kind>.jpg`
- 顯示於結局彈窗**最頂**、標題之上;構圖主體置中,四周可留氛圍。
- 出圖 prompt 見 `PROMPTS.md`。
